What is MkDocs?

MkDocs is a static site generator that's geared towards building project documentation. Documentation built with MkDocs is hosted on GitHub Pages, which is a free service that allows you to host your project's documentation directly from a GitHub repository. In other words, you write documentation in Markdown, and MkDocs builds the HTML files for you, and then you just push to GitHub.
